### 基本参数

| 参 数                | 规 格                                      |
| -------------------- | ------------------------------------------ |
| 工作电压             | 9.0-12.6v                                  |
| 马达类型             | 空心杯马达                                 |

| 有效角度（行程范围） | ±180°（单圈角度）\| ±368,640° （多圈角度） |
